# 🚔 Crime Safety Analytics Dashboard

An interactive Power BI dashboard designed to analyze crime incidents, victim demographics, and geographical crime distribution across different states and cities in the United States.

## 📊 Features

* Crime Type Analysis
* Victim Demographic Analysis
* State-wise Crime Distribution
* City-wise Crime Insights
* Victim Age Distribution
* Geographic Crime Mapping
* Interactive Filters and Slicers
* Data-Driven Visualizations

## 📈 Key Insights

* Analyzed crime frequency across different crime categories such as Assault, Burglary, Robbery, Theft, and Fraud.
* Identified crime distribution across multiple states and cities.
* Explored victim age patterns associated with different crime types.
* Visualized geographical crime occurrences using map-based analytics.
* Examined demographic characteristics of crime victims.

## 🛠️ Tech Stack

* Power BI
* Python
* Pandas
* SQL
* Data Visualization

## 📂 Dataset Attributes

* Crime Type
* Victim Age

* Victim Race
* City
* State
* Incident Information
